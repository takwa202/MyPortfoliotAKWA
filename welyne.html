Story

During my internship at Welyne, I spearheaded a transformative project focused on evolving their static website into a dynamic, feature-packed platform. This initiative involved overcoming the limitations of the existing site by introducing a robust back office, implementing user authentication, and enhancing overall functionality. The project aimed to elevate the user experience, optimize internal processes, and provide efficient content management, job application processing, and dashboard oversight. This endeavor underscored a commitment to innovation and modernization, propelling Welyne's digital presence to new heights.

Spotlight

.In crafting our welyne web app, we seamlessly combined Symfony's powerful PHP backend framework with a twin approach of HTML and CSS for the frontend. Symfony provided the robust backend structure, enabling efficient development of features like appointment scheduling and pharmacy integration. Meanwhile, the twin HTML and CSS strategy facilitated a responsive and visually appealing user interface, ensuring a clean, organized, and maintainable codebase. This collaborative synergy allowed us to deliver a user-friendly, comprehensive clinic management solution that seamlessly blends efficiency and aesthetics.
