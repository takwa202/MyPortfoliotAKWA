Story

I spearheaded the development of a feature-rich Campsite Manager app, revolutionizing the camping experience. This dynamic platform enables users to effortlessly book campsites, discover new locations by zone, and curate personalized wish lists. Noteworthy features include an intelligent group notification system, automating email alerts when campers gather at the same site. Beyond reservations, the app encompasses camper management, system reclamation, an integrated shop, and a social media component, fostering a sense of community. This project showcases a commitment to creating a holistic solution that seamlessly manages campsites, enhances user interactions, and elevates the overall camping experience.

Spotlight

I crafted the Campsite Manager app using Spring Boot for the backend and Angular for the frontend. Spring Boot streamlined backend development, handling features like booking management and user authentication. Angular, with its component-based structure, enabled the creation of a responsive and dynamic user interface. This powerful combination ensured a seamless user experience, blending robust backend functionality with an engaging frontend design.
